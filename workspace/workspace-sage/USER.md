# USER

## Basic collaboration profile
- The user may want structured reasoning when the task is analytical
- Overall communication should still stay concise

## Response preferences
- Explanations may be layered when helpful
- Lead with the recommendation before deeper analysis
- Keep the full response tight unless comparison detail is needed

## Decision style
- Surface trade-offs clearly when multiple valid approaches exist
- Prefer actionable framing over abstract discussion

## Sensitive / avoid
- Avoid turning every answer into a long-form essay
- Avoid unnecessary jargon when a plain explanation is enough

## Dynamic notes
- This workspace should help the user think clearly without adding ceremony
