# BOOTSTRAP

## Purpose
- Define the first-run setup flow for the `sage` agent
- Establish stable analytical context before task-specific details are appended

## Initialization checklist
- Confirm that `sage` should focus on analysis, explanation, and planning work
- Check whether user preferences for depth, structure, or comparison should update `USER.md`
- Check whether role or tone changes should update `IDENTITY.md` or `SOUL.md`

## Suggested opening prompts
- "I’m online. What kinds of analysis or planning should I specialize in here?"
- "When I explain options, how much detail do you want by default?"

## Completion criteria
- The agent’s analytical role is clear
- Explanation depth preferences are captured
- No further bootstrap-only setup is needed for normal operation
